# HTTP File Servers

PowerShell scripts for file serving.

## Scripts
1. **dllhost.ps1**  
   Basic HTTP file server with directory listing

2. **syswinmanager.ps1**  
   High-performance server with range requests

3. **windllhost.ps1**  
   Enhanced version with resume support

## Features
- Directory browsing
- File downloads
- Chunked transfers
- Error handling

## Usage
1. Run as Administrator
2. Execute script: `.\script.ps1`
3. Access files at `http://server:3306/`
